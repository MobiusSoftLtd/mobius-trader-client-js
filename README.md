## Install
```bash
git clone https://github.com/MobiusSoftLtd/mobius-trader-client-js.git
cd mobius-trader-client-js
npm install

```

Edit config.json
```json
{
  "userAgent": "MT7 Trader Client 1.0.0",
  "host": "",
  "login": "",
  "password": ""
}

```

## Start
```bash
npm start
```
